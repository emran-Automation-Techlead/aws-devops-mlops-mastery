How This Site Reaches You

1. S3 — The Warehouse

Every file here (this HTML, the CSS, any images) sits in an S3 bucket. S3 never goes down and scales to any amount of traffic without you doing anything.

2. CloudFront — The Delivery Trucks

Instead of every visitor fetching files from one place, CloudFront caches copies at edge locations near them — so someone in Tokyo and someone in Berlin both get a fast load.

3. Route 53 — The Address Book

Translates a human-readable domain name into the actual network location of the CloudFront distribution — the internet's GPS.

4. ACM — The Lock on the Door

Issues the SSL/TLS certificate so the connection is encrypted (https://) and browsers don't flag the site as unsafe.
